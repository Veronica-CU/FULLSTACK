***Experiment 1 - Simple Login Page using Context API React.***
The app conditionally renders pages based on the users authentication state. This is frontend only. 

***DESCRIPTION:***
Creates a Single Page Application,that allows users to login and logout.
Uses Context API to solve prop-drilling problem and uses global states to update the context across all the components. 

***TOOLS:*** 
React -Context API, Vite - SPA, JavaScript, CSS, Git &GitHub.


***FEATURES:***
- Login/logout page with username & password.
- Conditional rendering (Login/Private Page)
-  Logout functionality
-  Clean simple UI
-  Authentication handled using React Context API.
